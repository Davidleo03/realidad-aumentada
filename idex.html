Realidad Aumentada Facial

Experimenta ARCore en tu navegador

¿Cómo funciona?

  1. Asegúrate de estar en un dispositivo Android con ARCore compatible
  2. Haz clic en el botón "Activar AR" para iniciar la cámara
  3. Permite el acceso a la cámara cuando se te solicite
  4. Aparecerá un círculo rojo en tu frente entre las cejas
  5. Mueve tu cabeza para ver cómo el marcador se mantiene en posición

La vista de AR aparecerá aquí

Nota: Esta es una demostración visual. Para una implementación completa de ARCore con detección facial, se necesitaría integrar la API de ARCore y la API de detección facial de JavaScript.

Realidad Aumentada Facial

ARCore y detección facial en tiempo real

¿Cómo funciona?

  1. Asegúrate de estar en un dispositivo compatible (Android con ARCore o iOS con ARKit)
  2. Haz clic en el botón "Activar AR" para iniciar la cámara
  3. Permite el acceso a la cámara cuando se te solicite
  4. Aparecerá un círculo rojo en tu frente entre las cejas
  5. Mueve tu cabeza para ver cómo el marcador se mantiene en posición

Presiona "Activar AR" para comenzar

Compatibilidad

Esta aplicación funciona en dispositivos Android con ARCore y navegadores que soportan la API de detección facial. También funciona en iOS con Safari usando ARKit.

Nota: La aplicación utiliza la API de detección facial de JavaScript para identificar puntos faciales y posicionar el marcador en tiempo real.
